## Why

建立 AI 產業洞察系統的基礎開發環境，確保所有開發者使用統一的框架版本與 UI 組件庫，為後續的功能模組開發提供穩定的底層支持。

## What Changes

- 初始化 Next.js 14+ 專案骨架。
- 配置 Tailwind CSS 用於快速樣式開發。
- 集成 shadcn/ui 作為標準 UI 組件庫。
- 建立基礎目錄結構 (App Router)。

## Capabilities

### New Capabilities
- `project-foundation`: 建立可運行、具備基礎樣式與組件庫的 Next.js 環境。

### Modified Capabilities
- 無

## Impact

- **System**: 建立全新的專案根目錄結構。
- **Dependencies**: 安裝 `next`, `react`, `tailwind-css`, `lucide-react`, `clsx`, `tailwind-merge` 等基礎依賴。
